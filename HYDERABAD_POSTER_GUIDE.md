## The Contrast Zones Theme

The `contrast_zones` theme features:
- **High contrast** showing urban density
- **White background** (#FFFFFF)
- **Gradient of grays** for roads (from black motorways to lighter residential streets)
- **Darker in center, lighter at edges** - emphasizing the urban core
- **Minimalist aesthetic** - perfect for wall art

### Color Palette
- Background: `#FFFFFF` (white)
- Water: `#B0B0B0` (medium gray)
- Parks: `#ECECEC` (very light gray)
- Roads:
  - Motorways: `#000000` (black)
  - Primary: `#0F0F0F` (near black)
  - Secondary: `#252525` (dark gray)
  - Tertiary: `#404040` (medium dark gray)
  - Residential: `#5A5A5A` (medium gray)

## How to Generate the Hyderabad Poster

### Prerequisites

```bash
pip install -r requirements.txt
```

### Command

```bash
python3 create_map_poster.py -c "Hyderabad" -C "India" -t contrast_zones -d 18000
```

### Parameters Explained

- `-c "Hyderabad"` - City name
- `-C "India"` - Country name
- `-t contrast_zones` - Theme (high contrast minimalist style)
- `-d 18000` - Distance radius in meters (18km)
  - This captures the full city center including:
    - Charminar and Old City
    - Hussain Sagar Lake
    - Hitech City
    - Gachibowli
    - Banjara Hills
    - Secunderabad

### Output

The poster will be saved in the `posters/` directory with the format:
```
hyderabad_contrast_zones_YYYYMMDD_HHMMSS.png
```

## Alternative Distance Values

You can adjust the `-d` parameter to focus on different areas:

| Distance | Coverage Area |
|----------|---------------|
| 8000m | Old City + Charminar area |
| 12000m | Central Hyderabad including Hussain Sagar |
| 18000m | Full metro including Hitech City (Recommended) |
| 25000m | Greater Hyderabad with suburbs |

## Network Requirements

The script requires internet access to:
1. **OpenStreetMap Overpass API** (`overpass-api.de`) - for downloading map data
2. **Nominatim** (`nominatim.openstreetmap.org`) - for geocoding (now optional with fallback)

If you encounter proxy or network issues, ensure these domains are accessible.
