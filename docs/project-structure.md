# Project Structure

```
map_poster/
├── create_map_poster.py          # Main script
├── themes/               # Theme JSON files
├── fonts/                # Roboto font files
├── posters/              # Generated posters
└── README.md
```

## Directory Descriptions

- **`create_map_poster.py`**: Main Python script that generates map posters
- **`themes/`**: Contains JSON theme configuration files
- **`fonts/`**: Font files used for text rendering (Roboto family)
- **`posters/`**: Output directory where generated poster images are saved
- **`cache/`**: Cache directory for storing geocoding and map data
